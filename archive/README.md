# archive ガイド

`archive/` は、With ポータルへ方向転換する前の投票プロジェクト資産を退避した場所です。

## 含まれるもの

- `v1-vote-result/`: 旧公開ページ
- `html_experiment/`: Firebase 前提の投票プロトタイプ
- `specs/`: 旧仕様書
- `firebase/`: 旧 Firebase 設定
- `firebase-survey-results-2026-04-04.json`: 旧集計データ

## 扱い方

- ここにあるものは履歴確認用です
- 現在の実装判断は `../specs/` と `../site/` を正史として扱います
- 将来再利用する場合も、現行方針へ合わせて再設計してから使います
