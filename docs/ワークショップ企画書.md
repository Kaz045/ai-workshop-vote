# 【企画案】AIブームに惑わされない：AI時代の仕事のリアル  〜自分のコアバリューをAIで拡張する仕事術〜

## ワークショップ概要

AIツール、AI開発、企業AI、法人営業の5つのテーマから
**「AIが仕事をどう変えるのか」** を構造的に整理するワークショップ。

現在、AIブームの中では

- AI副業
- AIで稼ぐ
- AIコンサル

といった様々な情報が溢れています。

しかし多くの場合

**「自分のコアがないままAIを始めようとしている」**

ケースが少なくありません。

私も安直に「AI副業で稼ぐ」というキャッチに捕まってしまった一人です。

本ワークショップでは

**AIで何を始めるか** ではなく

**自分のコアバリューをAIでどう増幅させるか**

という視点からAIを整理します。

---

## メッセージ

**「世の中のAIブームに惑わされるな」**

AIは

新しい人生を作るツールではなく

**自分の価値を増幅させるツール**

です。

AIで新しいことを無理に始める必要はありません。

むしろ重要なのは

- 今自分が持っているスキル
- 今やっている仕事
- 自分が価値を出している領域

を起点に

**AIを掛け合わせること** です。

---

## このワークショップで扱うテーマ

本ワークショップでは

1. AIを日常のOSとして使う（ChatGPT / Claude / Gemini — 基本AIの育て方）
2. AIツールによる効率化・ツール開発（バイブコーディング入門）
3. AI駆動開発；非エンジニアとエンジニアの境目（AI駆動開発：VS Code / Cursor / Anti-gravity）
4. Middle〜Enterprise企業 AI導入の一面（Copilotの有用性）
5. AI時代に価値が残る仕事：法人営業 × AI（営業とは｜営業のAI活用）

という流れで

**AIと仕事の関係を構造的に整理**します。
