# データモデル

## 1. 会員向けページ保護の設定

現在の会員向けページは、HTML 属性と JavaScript で軽量なページ保護を実装しています。

### エンティティ: `ProtectedPage`

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| `gateKey` | string | 必須 | `sessionStorage` に保存するキー |
| `passwordHash` | string | 必須 | SHA-256 ハッシュ値 |
| `contentSelector` | string | 必須 | 保護対象コンテンツのセレクタ |

## 2. 第1回アーカイブの表示データ

現在の会員向けアーカイブは、静的な HTML として次の情報を表示します。

### エンティティ: `WorkshopArchive`

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| `slug` | string | 必須 | ページ識別子 |
| `title` | string | 必須 | 回のタイトル |
| `summary` | string | 必須 | ページ冒頭の説明 |
| `insights` | string[] | 必須 | 振り返りポイント |
| `nextSteps` | string[] | 必須 | 次回へつなげる改善点 |

## 3. 投票結果の静的データ

第1回ページでは、旧投票結果を静的な表として再掲しています。

| テーマ | 票数 |
|---|---|
| AIでツールを作る | 11 |
| AI開発の世界を覗く | 7 |
| ChatGPTなどを仕事の相棒として使う | 4 |
| 企業でのAI導入とMS Copilot | 1 |

## 4. 運用ルール

- 現在のポータルでは、データベースや API は使わない
- 新しい回を追加するときは、静的ページとして情報を保持する
- 原本 PDF や動画などは `records/` または外部ストレージで管理する
